# AWS Troubleshooting Hub

A React-based troubleshooting platform for AWS services with organized access to troubleshooting guides across multiple service categories.

## Quick Start

```bash
npm install
npm start
```

## Features

- Multi-service support for 11+ AWS service categories
- Responsive design for all devices
- Advanced search functionality
- Interactive navigation with visual cards

## Service Categories

- **DMS** - Database Migration Service
- **Networking** - VPC, Load Balancer, networking solutions
- **Database** - RDS, Aurora, database services
- **Analytics** - Data analytics and processing
- **Big Data** - Large-scale data processing
- **Security** - IAM, encryption, security services
- **Linux/Windows** - System administration
- **SVLS** - Serverless (Lambda, API Gateway, Step Functions)
- **DMI** - Developer Mobile, Messaging & IoT
- **ELB** - Elastic Load Balancer
